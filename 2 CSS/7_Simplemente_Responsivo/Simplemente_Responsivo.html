@media queries

CSS uses the @media rule to detect information about the device being used to acces the website. For example, it can be used to detected the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but keep in mmind that it is capable of much more.

Desktop:Width greater than or equal to 1024px

Tablet:Width between 481px and 1023px

Phone:Width leess than or equal to 480px
